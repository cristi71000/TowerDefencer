## Recommended Replacements

For a more polished look, consider replacing these placeholder icons with assets from:

- **Kenney.nl** - https://kenney.nl/assets (CC0 License)
  - UI Pack (430+ sprites)
  - Game Icons
  - Game Icons Expansion

- **Game-icons.net** - https://game-icons.net/ (CC BY 3.0)
  - Thousands of game icons

- **Unity Asset Store** - Free UI asset packs available

When replacing assets, ensure you:
1. Update the sprite references in `UIIconsData.asset`
2. Update the TowerData ScriptableObjects with new tower icons
3. Document new licenses in this file
